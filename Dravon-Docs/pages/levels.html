Level System

Engaging XP and leveling system with rewards, leaderboards, and community competition

Setup Guide

1

Enable Level System

Activate the leveling system to start tracking user activity and XP.

>levels enable
# Enables XP tracking for all server members
# Users gain XP by sending messages and being active
2

Configure XP Settings

Customize XP gain rates and cooldowns to match your server's activity.

>levels config xp 15-25
# Set XP range per message (15-25 XP)

>levels config cooldown 60
# Set cooldown between XP gains (60 seconds)
3

Set Up Level Rewards

Create role rewards for reaching specific levels to motivate users.

>levels reward add 5 @Active Member
>levels reward add 10 @Regular
>levels reward add 25 @Veteran
# Users automatically get roles when reaching levels

Level Progression System

XP Calculation

The XP required for each level increases progressively:

XP Required = Level² × 100
Level 1 100 XP
Level 5 2,500 XP
Level 10 10,000 XP
Level 25 62,500 XP

XP Sources

Users can gain XP through various activities:

  • Sending messages (configurable XP)
  • Voice channel activity
  • Playing games with bot
  • Daily bonuses
  • Special events

Pro Tips

Balance XP Rates

Set appropriate XP ranges to maintain engagement without making progression too easy or difficult.

Meaningful Rewards

Create role rewards that provide actual benefits like access to special channels or permissions.

Cooldown Settings

Use appropriate cooldowns (30-120 seconds) to prevent spam while allowing natural conversation flow.

Monitor Activity

Regularly check leaderboards and adjust settings based on your server's activity patterns.

Community Events

Host XP boost events or competitions to increase engagement and community participation.

Prevent Abuse

Monitor for XP farming and adjust settings or manually moderate if necessary to maintain fairness.
