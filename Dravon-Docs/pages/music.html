Music Player

High-quality music streaming with multi-platform support and interactive controls

Setup Guide

1

Join Voice Channel

Join a voice channel and use the play command to start streaming music.

# Join a voice channel first, then:
>play Never Gonna Give You Up
# Dravon will automatically join and start playing
2

Premium Setup (Optional)

Premium users can connect Spotify for direct playlist streaming.

# Premium users only:
>spotify connect
# Follow the authentication process
# Then use: >play spotify:playlist:your_playlist_id
3

Configure DJ Role (Optional)

Set up a DJ role to restrict music controls to specific users.

>music djrole @DJ
# Only users with DJ role can control music
# Use "none" to allow everyone

Commands List

Playback Control

>play <song/url>

Play a song from YouTube, Spotify, or SoundCloud

>pause

Pause the current song

>resume

Resume playback

>stop

Stop playback and clear queue

Queue Management

>skip

Skip to the next song

>previous

Go back to previous song

>queue

View the current music queue

>clear

Clear the entire queue

Playback Modes

>shuffle

Toggle shuffle mode on/off

>repeat

Toggle repeat mode (off/song/queue)

>autoplay

Toggle autoplay similar songs

Audio Control

>volume <1-100>

Set playback volume

>bass <-5 to 5>

Adjust bass levels

>nightcore

Toggle nightcore effect

>8d

Toggle 8D audio effect

Playlist Features

>playlist create <name>

Create a new playlist

>playlist add <name> <song>

Add song to playlist

>playlist play <name>

Play entire playlist

>playlist list

View all your playlists

Premium Features

>spotify connect

Connect your Spotify account (Premium only)

>spotify playlists

View your Spotify playlists

>play spotify:<uri>

Play directly from Spotify

>247

Enable 24/7 mode (Premium only)
