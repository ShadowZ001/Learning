Commands List

Setup Commands

>ticket panel <channel>

Create ticket panel in specified channel

>ticket category add <name> <emoji>

Add new ticket category

>ticket role add <role>

Add support role for ticket access

>ticket logs <channel>

Set channel for ticket logs

Ticket Management

>ticket close [reason]

Close the current ticket

>ticket reopen

Reopen a closed ticket

>ticket claim

Claim ticket for personal handling

>ticket unclaim

Remove claim from ticket

User Management

>ticket add <user>

Add user to current ticket

>ticket remove <user>

Remove user from ticket

>ticket transfer <user>

Transfer ticket ownership

Transcripts

>ticket transcript

Generate ticket transcript

>ticket save

Save ticket messages to file

>ticket export

Export ticket data as JSON

Information

>ticket info

View current ticket information

>ticket list

List all open tickets

>ticket stats

View ticket system statistics

Advanced Settings

>ticket autoclose <time>

Set auto-close timer for inactive tickets

>ticket limit <number>

Set max tickets per user

>ticket naming <format>

Set ticket channel naming format

Ticket Workflow

1

User Creates Ticket

User clicks category button on ticket panel to create a new support ticket.

2

Channel Creation

Bot creates private channel with user and support roles, sends welcome message.

3

Support Interaction

Support staff can claim ticket, add/remove users, and provide assistance.

4

Resolution & Closure

Once resolved, ticket is closed with transcript saved and channel archived.
