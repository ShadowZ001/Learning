Frequently Asked Questions

Find instant answers to the most common questions about Sonic Bot setup, features, and troubleshooting.

How do I add Sonic to my Discord server? +

Click the "Add to Discord" button on our website, select your server, and authorize the required permissions. You need "Manage Server" permissions to add bots. The setup wizard will guide you through the initial configuration automatically.

What permissions does Sonic need and why? +

Sonic requires Administrator permissions to provide comprehensive server protection and management. This includes managing roles for AntiNuke protection, moderating channels for AutoMod, managing members for security features, and accessing audit logs for threat detection. All permissions are used responsibly and transparently.

How do I configure AntiNuke protection? +

Use /antinuke setup to enable protection with default settings, or /antinuke config for advanced configuration. You can set thresholds for different actions, configure whitelist users, and customize punishment actions. The system includes protection against mass bans, kicks, channel deletion, role manipulation, and webhook abuse.

How does the AutoMod system work? +

Sonic's AutoMod uses advanced algorithms to detect spam, inappropriate content, and rule violations. Use /automod setup to get started with preset configurations, then customize with /automod filter to add custom words, /automod spam for spam detection, and /automod logs to set up logging channels.

Is Sonic completely free to use? +

Yes! Sonic is 100% free with all 180+ features available to every server. There are no premium tiers, subscription fees, or hidden costs. We believe in providing powerful Discord server management tools accessible to everyone, from small communities to large servers.

How do I set up Join-to-Create voice channels? +

Use /j2c setup to create a Join-to-Create channel. When users join this channel, Sonic automatically creates a private voice channel for them. Configure channel names, user limits, and permissions with /j2c config. Channels are automatically deleted when empty.

Can I customize commands and prefixes? +

Sonic supports both slash commands (/) and traditional prefix commands (!). You can change the prefix using /settings prefix. Many commands have aliases and can be customized. You can also disable specific commands or modules using /settings disable for granular control.

How do I report bugs or request features? +

Report bugs and request features through our GitHub repository for detailed tracking, or join our Discord server for immediate assistance. Include server ID, command used, error messages, and steps to reproduce for bugs. Feature requests are reviewed by our development team and community.

What makes Sonic different from other Discord bots? +

Sonic combines military-grade security, intelligent automation, and comprehensive server management in one bot. Unlike other bots that specialize in single areas, Sonic provides 180+ features including advanced AntiNuke protection, AI-powered moderation, entertainment systems, analytics, and automation - all completely free with 99.9% uptime.

How do I get help if I'm still having issues? +

Join our Discord server for real-time help from our community and support team. Use /help in your server for command-specific assistance. Check our documentation for detailed guides, or contact our priority support team for urgent issues. We're here to help 24/7!
